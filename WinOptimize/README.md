# Windows Optimization v1.0
### by Marcelo Borasi — 07/26

Optimizacion segura y reversible para Windows 10 Pro x64 (hogar/gaming).
Un boton hace todo, un boton lo deshace todo. / Safe, reversible optimization
for Windows 10 Pro x64 (home/gaming). One button does everything, one button
undoes everything.

---

## 🇦🇷 Español

**Que hace:** quita apps innecesarias, desactiva telemetria, activa Modo
Juego y Rendimiento Maximo, revisa tus discos (SSD/HDD/NVMe), verifica la
integridad del sistema, y mas — todo detectado automaticamente, todo
reversible. Impresora, red compartida, audio, Firewall, Defender y Windows
Update **siempre protegidos**.

**Como usarlo:**
1. Ir a [Releases](../../releases) y descargar el `.zip` de la ultima version.
2. Extraer la carpeta completa (no mover archivos sueltos).
3. Clic derecho sobre `WinOptimize.bat` → **Ejecutar como administrador**.
4. Elegir **[1] OPTIMIZAR TODO** en el menu.

Manual completo, con todos los detalles y el descargo de responsabilidad,
en [`README.txt`](README.txt).

**Requisitos:** Windows 10 Pro x64 (ver compatibilidad con otras ediciones
en el manual completo).

## 🇺🇸 English

**What it does:** removes unneeded apps, disables telemetry, enables Game
Mode and Ultimate Performance, checks your disks (SSD/HDD/NVMe), verifies
system integrity, and more — all auto-detected, all reversible. Printer,
LAN sharing, audio, Firewall, Defender and Windows Update are **always
protected**.

**How to use it:**
1. Go to [Releases](../../releases) and download the latest `.zip`.
2. Extract the whole folder (don't move individual files).
3. Right-click `WinOptimize.bat` → **Run as administrator**.
4. Choose **[1] OPTIMIZE EVERYTHING** from the menu.

Full manual, with all details and the liability disclaimer, in
[`README.txt`](README.txt).

**Requirements:** Windows 10 Pro x64 (see compatibility with other
editions in the full manual).
